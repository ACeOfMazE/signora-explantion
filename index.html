Signora: A Sign Language to Speech Translator

1. Introduction

Millions rely on sign language, but most non-signers canâ€™t understand it. Signora bridges this gap using Arduino Nano and 5 flex sensors to translate hand gestures into speech using Python.

2. Problem Statement

  • Communication barrier for hearing/speech impaired people.
  • Lack of real-time, low-cost gesture-to-voice solutions.
  • Commercial tools are expensive or language-limited.

3. Objective

  • Detect gestures via flex sensors on glove.
  • Interpret signals using Python.
  • Convert gestures into readable and audible output.
  • Operate on basic hardware (Arduino + PC).

4. Components Used

Hardware:

  • Arduino Nano
  • 5 Flex Sensors
  • Resistors, Breadboard, Jumper Wires
  • Glove
  • USB Cable

Software & Libraries:

  • Python
  • serial
  • pyttsx3
  • time, threading, os, logging, collections

5. How the System Works

  • Flex Sensors: Detect finger bending resistance.
  • Arduino: Reads values, sends data over USB.
  • Python: Matches patterns, prints and speaks gesture text using pyttsx3.

6. Example Gesture: "HELLO"

  • Thumb: bent (value > 90)
  • Index, Middle: straight
  • Ring, Pinky: bent
  • Recognized and spoken by the system.

7. Current Limitations

  • Arduino Nano has limited power and ports.
  • Does not support dynamic/moving gestures.
  • False triggers possible due to noise.

8. Achievements & Impact

  • Supports communication for non-verbal users.
  • Offline and affordable.
  • Useful in schools, public counters, hospitals.

9. Future Improvements

  • Use ESP32/Raspberry Pi for better performance & Wi-Fi.
  • ML-based dynamic gesture recognition.
  • Android/iOS app with Bluetooth support.
  • Camera-based gesture recognition.

10. Real-World Use Cases

  • Inclusive education for deaf students.
  • Assistive communication in healthcare.
  • Improved accessibility in public services.

11. Conclusion

Signora shows that affordable tech can solve big problems. With simple hardware and smart software, it gives voice to gestures and hope to the voiceless.
